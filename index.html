STUDENT MANAGEMENT CRUD SYSTEM

This project is a Python-based CRUD application designed to manage student records efficiently. It allows users to create, read, update, and delete student information, storing data in both JSON and CSV formats for easy backup and data portability. The system provides a simple console interface and demonstrates essential skills in file handling, data manipulation, and structured programming.

MINI OS

This project is a mini operating system built using HTML and CSS. It simulates a desktop interface with multiple apps, each displayed inside separate iframes. The layout includes icons, windows, and basic UI elements that mimic a real OS environment. Users can open different “apps” such as a browser, notes, or settings, all running within interactive iframe windows. The design focuses on simplicity, smooth navigation, and a clean, modern visual style.
